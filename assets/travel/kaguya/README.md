# 超かぐや姫 週末巡礼写真置き場

このフォルダは、当日自分たちで撮影した写真を後から差し込むための場所です。

現時点では `travel/kaguya-tachikawa-weekend.html` に存在しない画像への `img` タグは置いていません。画像を追加するときは、HTML内の `<!-- 後で自分たちで撮影した写真をここに追加する -->` コメント付近に差し込んでください。

ファイル名の例:

- `yokohama-tammachi.jpg`
- `shibuya-shimokita.jpg`
- `roppongi-akasaka.jpg`
- `tachikawa-south.jpg`
- `tachikawa-north-greensprings.jpg`
- `sunagawa-nishimatsuya.jpg`
- `haneda-airport.jpg`

運用メモ:

- 第三者サイトの画像、スクリーンショット、作品画像、アニメキャプチャは入れない。
- 自分たちで撮った写真だけを置く。
- 人の顔、表札、車のナンバーなどが写っている場合は公開前に加工する。
- GitHub Pagesで公開する可能性があるため、権利関係が曖昧な画像は使わない。
